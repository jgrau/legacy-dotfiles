# Dotfiles

## Manual steps

### Gitx

* Open `GitX`
* Go to menu `GitX` -> `Enable Terminal Usage`

### Caps Lock -> CTRL

* Open `Keyboard Preferences`
* Cick `Modifier Keys...`
* Select `^ Control` as the `Caps Lock` key

### Change Shell

* `sudo vim /etc/shells`
* Add `/usr/local/bin/zsh`
* `chsh -s /usr/local/bin/zsh`
